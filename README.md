# tiwtter-backend

### Run locally 

Clone the project

```bash
  https://github.com/Bhargavi952/tiwtter-backend.git
```
Go to the project directory

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm run start
```
  
  
## Tech Stack

- Server:  Node, Express

- DataBase: MongoDB

  
## Features

- Authentication
- Tweet Post
